EMG-Based Human Machine Interface

Code

Raspberry Pi 3B

All code running on the Raspberry Pi was written in Python 2.7.14. The basis of the bluetooth communication for the Myo Armband was developed by Fernando Cosentino (PyoConnect_v2.0).

MATLAB

MATLAB r2017b was used to create figures and to explore the efficacy of a trained neural network for gesture recognition.
